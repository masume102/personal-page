پروژه وب‌سایت شخصی
---------------------

این پروژه شامل یک وب‌سایت شخصی است که با استفاده از HTML و CSS طراحی شده و با GitHub Actions روی GitHub Pages مستقر شده است. پروژه حداقل ۵ بار با تغییرات مختلف مستقر شده است.

### اطلاعات پروژه
- نام پروژه: صفحه شخصی من
- مخزن GitHub: https://github.com/masume102/personal-page.git
- لینک GitHub Pages: https://github.com/masume102/personal-page.git
- فایل‌ها:
  - index.html: ساختار اصلی وب‌سایت
  - styles.css: استایل‌های وب‌سایت
  - .github/workflows/deploy.yml: فایل تنظیمات GitHub Actions برای استقرار
  - README.md: مستندات پروژه

### تاریخچه استقرار
1. استقرار اولیه: ایجاد ساختار پایه وب‌سایت
2. استقرار دوم: تغییر تم رنگی (هدر، فوتر، و مهارت‌ها)
3. استقرار سوم: افزودن تصویر پروفایل به بخش درباره من
4. استقرار چهارم: افزودن پروژه‌های بیشتر به بخش پروژه‌ها
5. استقرار پنجم: افزودن انیمیشن به لینک‌های ناوبری

### دستورالعمل‌های بررسی
1. لینک GitHub Pages (https://USERNAME.github.io/personal-page/) را در مرورگر باز کنید تا وب‌سایت را مشاهده کنید.
2. به مخزن GitHub (https://github.com/USERNAME/personal-page) مراجعه کنید و تب Actions را بررسی کنید تا تاریخچه ۵ استقرار موفق را ببینید.
3. فایل README.md شامل جزئیات کامل پروژه است.
